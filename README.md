# 🚀 React Job Card UI

A modern **Job Listing Card UI** built using **React.js** demonstrating core React concepts like **Components, Props, and Array Mapping**.

This project displays job openings from top tech companies and other big tech giants in a clean and responsive card layout.

---

## 📌 Project Overview

This application renders multiple job cards dynamically using an array of job objects. Each job listing contains company information, role details, salary, job type, and location.

The project focuses on understanding reusable components and passing data efficiently using props.

## ⚙️ Concepts Used

### 🔹 Components

Created a reusable `JobCard` component to display job information.

### 🔹 Props

Data such as company name, logo, job role, salary, and location are passed using props.

### 🔹 Array Mapping

Job listings are dynamically rendered using:

```javascript
jobs.map((job) => (
  <JobCard {...job} />
))
```
